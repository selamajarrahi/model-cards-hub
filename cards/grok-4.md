# Grok 4

**Organization:** xAI  
**Release Date:** Late 2025  
**Type:** Large Language Model  
**Access:** [x.ai](https://x.ai)

---

## Overview

Grok 4 is xAI's flagship model, known for real-time information access via X (Twitter), irreverent personality, and strong reasoning capabilities. Built to be "maximally truthful" with less restrictive safety filters.

**Key Claims:**
- Real-time access to X/Twitter data
- Competitive with GPT-5.2 and Claude Opus 4
- Less restrictive than competitors on certain topics
- Native image generation (Aurora)

## Unique Features

### 1. Real-Time X Integration
Grok can search and cite real-time posts from X (Twitter), giving it access to breaking news and current discussions.

### 2. "Fun Mode"
Optional personality mode with humor and less formal responses. Will engage with edgier topics.

### 3. Aurora Image Generation
Built-in image generation capability, competing with DALL-E and Midjourney.

### 4. DeepSearch
Multi-step research capability that synthesizes information from multiple sources.
